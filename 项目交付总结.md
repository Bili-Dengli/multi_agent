## 🎯 使用流程速查表

### 最快开始方式（3 分钟）

```bash
# 1. 安装
cd openclaw-skillspack/skills/multi_agents
./install.sh

# 2. 测试
python3 scripts/task_update.py create TASK-TEST-001 "测试任务"
python3 scripts/task_update.py done TASK-TEST-001 "完成"

# 3. 验证
cat ~/.openclaw/workspace/multi_agents/data/tasks.json
```

### 完整协作流程（典型 30-60 秒）

```
用户 → Director（创建任务）
    → Manager（拆解子任务）
    → Worker（执行任务）
    → Manager（汇总结果）
    → Director（回复用户）
```
